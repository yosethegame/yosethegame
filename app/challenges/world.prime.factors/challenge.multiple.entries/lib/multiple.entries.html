Update your server to make it accept a GET request with several numbers and make it respond a JSON containing the decompositions of each numbers.

For example, receiving

http://your-server/primeFactors?number=300&number=120&number=hello

You should respond

[
    {
        "number" : 300,
        "decomposition" : [ 2, 2, 3, 5, 5 ]
    },
    {
        "number" : 120,
        "decomposition" : [ 2, 2, 2, 3, 5 ]
    },
    {
        "number" : "hello",
        "error" : "not a number"
    },
]

Your server will be called with several numbers.
